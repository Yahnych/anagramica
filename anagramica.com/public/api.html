ANAGRAMICA

API

Hi There! Anagramica offers an API! This API lets you:
  • Get a best case anagram for a given letter set
  • Get all anagrams for a given letter set
  • See if a word exists in the dictionary
  • Wrap results in a JSONP callback

/best (GET)

http://www.anagramica.com/best/:letters

Returns:
   {best:[<word1><,word2><,word3><...>]}

/all (GET)

http://www.anagramica.com/all/:letters

Returns:
   {all:[<word1><,word2><,word3><...>]}

/lookup (GET)

http://www.anagramica.com/lookup/:word

Returns:
   {found:<1|0>}

JSONP

?callback=<callback>

Returns:
   <callback>(<results>);
